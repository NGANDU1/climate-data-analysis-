## 🎯 Chart Types Explained

### 1. Line Chart - Temperature Trends
**Purpose:** Compare temperature across provinces over time

**Features:**
- ✅ Multiple datasets (Week 1, 2, 3)
- ✅ Smooth curves
- ✅ Filled area under lines
- ✅ Interactive hover tooltips
- ✅ Click legend to hide/show lines

**Colors Used:**
- Dark Green line = Week 1
- Light Green line = Week 2  
- Blue line = Week 3

---

### 2. Pie Chart - Risk Distribution
**Purpose:** Show percentage of regions at each risk level

**Features:**
- ✅ 4 colored segments
- ✅ Automatic percentage calculation
- ✅ Legend with labels
- ✅ Visual impact for presentations

**Segments:**
- 🟢 Green = 40% Low Risk (4 regions)
- 🟡 Yellow = 30% Medium Risk (3 regions)
- 🟠 Orange = 20% High Risk (2 regions)
- 🔴 Red = 10% Critical (1 region)

---

### 3. Bar Chart - Rainfall Comparison
**Purpose:** Compare monthly rainfall across regions

**Features:**
- ✅ Vertical bars
- ✅ Rounded corners
- ✅ Y-axis in millimeters
- ✅ Easy visual comparison

**Data Example:**
- Lusaka: 145mm (tallest bar)
- Copperbelt: 98mm
- Eastern: 167mm (highest)
- Northern: 76mm (lowest)

---

### 4. Doughnut Gauge - Humidity Levels
**Purpose:** Show humidity distribution in gauge format

**Features:**
- ✅ Hollow center (70% cutout)
- ✅ 3 color zones
- ✅ Minimalist design
- ✅ Percentage in center

**Zones:**
- 🟢 Green = Normal (40-60%) - 65%
- 🟠 Orange = Elevated (60-80%) - 25%
- 🔴 Red = High (>80%) - 10%

---

### 5. Doughnut Chart - Disaster Types
**Purpose:** Analyze types of disasters detected

**Features:**
- ✅ 5 categories
- ✅ Different colors
- ✅ Right-side legend
- ✅ 60% cutout ratio

**Categories:**
- 🔵 Blue = Flood (35%)
- 🟡 Yellow = Drought (25%)
- 🟠 Orange = Extreme Heat (20%)
- 🟣 Purple = Storm (12%)
- 🟢 Green = General (8%)

---

### 6. Dual-Axis Line Chart - Wind & Pressure
**Purpose:** Monitor two different metrics simultaneously

**Features:**
- ✅ Left Y-axis = Wind Speed (km/h)
- ✅ Right Y-axis = Pressure (hPa)
- ✅ Two independent lines
- ✅ Filled areas
- ✅ 7-day timeline

**Example Reading:**
- Monday: 15.2 km/h wind, 1015 hPa pressure
- Thursday: 22.3 km/h wind, 1012 hPa pressure

## 🎨 Customization Options

### Change Chart Colors

Edit in `analytics.html`:
```javascript
const colors = {
    emerald: '#10b981',      // Main green
    emeraldLight: '#34d399', // Light green
    emeraldDark: '#059669',  // Dark green
    blue: '#0ea5e9',         // Blue accent
    orange: '#f97316',       // Orange accent
    red: '#dc2626',          // Red accent
};
```

### Adjust Chart Sizes

Modify height in CSS:
```css
.chart-container {
    height: 300px;  /* Standard chart */
}

.chart-container-large {
    height: 400px;  /* Large chart */
}
```

### Add More Data Points

Update datasets in JavaScript:
```javascript
datasets: [{
    label: 'Week 4',
    data: [28.1, 26.8, 27.9, ...],  // Add 10 values
    borderColor: '#a7f3d0',         // Mint color
    tension: 0.4,
    fill: true
}]
```

## 💡 Pro Tips

### For Presentations

1. **Fullscreen Mode** (F11 browser)
   - Shows charts in high detail
   - Professional appearance
   - No distractions

2. **Focus on Specific Charts**
   - Click other legend items to gray out
   - Highlight one dataset at a time
   - Direct attention effectively

3. **Export Screenshots**
   - Right-click chart → Save as PNG
   - Use in reports/presentations
   - Share with stakeholders

### For Daily Monitoring

1. **Check Active Alerts First**
   - Top stats row shows current count
   - Red/orange numbers need attention

2. **Review Temperature Trends**
   - Look for unusual patterns
   - Compare week-over-week changes
   - Identify heating/cooling trends

3. **Monitor Rainfall**
   - High bars = flood risk
   - Low bars = drought concern
   - Regional comparisons important
